If you have node.js installed, best way to run the sample is to use Node's http-server.
Running html from local filesystem can result CORS denials.

1) install http-server to node.js if needed by running "npm install http-server -g"
2) Navigate to directory where project files were extracted
3) run "http-server" command
4) Open browser using one of the URLs displayed in console


Directly from local drive
1) Extract all files from zip to a folder
2) Double-click index.html
